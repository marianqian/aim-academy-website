Welcome to the AIM Academy

The AIM (Artificial Intelligence Mastery) Academy, part of the organization Humanitas4All, provides resources to teach machine learning and computer science concepts using interactive coding exercises and lectures. (about us)

Humanitas4All serves to improve the education of low-income communities through tutoring sessions and summer camps. We focus on the computer science sector by collecting resources and creating materials for students to access. (learn more about Humanitas4All http://www.humanitas4all.org/)
